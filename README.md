# DHEALTH- A Distributed Privacy-Preserving Framework for Smart Healthcare Systems
DHEALTH is a distributed healthcare system designed to securely manage and share patient data using privacy-preserving techniques.

## Features
- Secure user authentication
- Encrypted data storage
- Role-based access control
- Patient and doctor data management

## Tech Stack
- Java (JSP/Servlets)
- MySQL
- HTML, CSS, JavaScript

## How to Run
1. Import project into NetBeans/Eclipse
2. Setup MySQL database
3. Configure server (Tomcat)
4. Run the project
